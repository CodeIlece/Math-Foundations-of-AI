  2. MF-5 (Дополнительные главы математической теории ИИ): Способен применять продвинутые математические методы (теория графов, функциональный анализ, топология, теория категорий) для решения сложных задач ИИ и разработки новых алгоритмов.

6. ТЕМАТИКА КУРСОВЫХ РАБОТ
Примерная тематика отражает интеграцию индикаторов MF-3, LLM-5 и O-3:
  1. Разработка дифференцируемого слоя MILP-солвера для маршрутизации запросов внутри трансформерной архитектуры. Цель: снижение задержек инференса путем оптимального распределения внимания между экспертами MoE-модели.
  2. Система автоматизированного инжиниринга знаний на базе связки GigaChat + PuLP. Создание конвейера, где языковая модель извлекает ограничения из ГОСТов или ТУ, строит задачу линейного программирования для расчета рецептуры продукта, а результат автоматически валидируется на непротиворечивость корпоративной онтологии (O-3).
  3. Применение генетических алгоритмов для топологической оптимизации структуры графа знаний под специфические паттерны SPARQL-запросов. Минимизация времени обхода графа при сохранении полноты ответа.
  4. Байесовская оптимизация гиперпараметров физически-информированных нейросетей (PINN) с учетом жестких физических ограничений.
