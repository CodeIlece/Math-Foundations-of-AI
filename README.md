# Математические основы ИИ

## Состав команды

Бакулева М. А., Калитвин В. А., Ким С. Е., Пономарев Д. А.,
Попков Р. А., Шильников К. Е.

## Цель дисциплины

Сформировать у студентов фундаментальные математические знания и практические навыки, необходимые для решения прикладных задач машинного обучения, включая разработку, оценку и оптимизацию моделей, в соответствии с требованиями к роли «Инженер машинного обучения» (ML Engineer) на уровне «Средний» (С) согласно КРМ ИИ 3.0.

## Задачи дисциплины

1. Систематизировать и углубить знания линейной алгебры и математического анализа в контексте их применения в задачах оптимизации и обучения моделей ИИ.
2. Освоить современные методы оптимизации (выпуклой и невыпуклой) и научиться применять их для обучения моделей и настройки гиперпараметров.
3. Сформировать практические навыки применения статистических методов для анализа данных, проверки гипотез, валидации моделей и оценки их качества.
4. Развить умение интерпретировать результаты работы моделей и обосновывать принятые инженерные решения с точки зрения математической корректности.

## Пререквизиты

Для успешного освоения курса «Математические основы ИИ» студент должен владеть следующими компетенциями и знаниями на уровне, соответствующем завершенным дисциплинам бакалавриата (МF-1.2 Б, МF-1.3 Б, МF-2.1 Б, МF-6.1 Б, МF-6.2 Б, PL-1.1 Б):

1. Математический анализ: Владение понятиями предела, производной, интеграла, ряда. Умение исследовать функции одной и нескольких переменных.
2. Линейная алгебра: Знание основ матричного исчисления, операций над векторами и матрицами, понятие линейной (не)зависимости, собственных значений и векторов.
3. Теория вероятностей и математическая статистика: Понимание основ теории вероятностей, законов распределения случайных величин, основных понятий математической статистики (оценки, проверка гипотез).
4. Программирование: Базовые навыки программирования на языке Python (синтаксис, работа с типами данных, написание функций, использование циклов и условий).
5. Логика и дискретная математика: основы булевой алгебры, логика высказываний, кванторы, элементы теории множеств, теория графов.

## Компетенции

**МF-3** Способен применять современные методы оптимизации для обучения моделей машинного обучения, настройки гиперпараметров и решения задач искусственного интеллекта;

**МF-4** Способен применять статистические методы для анализа данных, валидации моделей машинного обучения и проведения экспериментов в области ИИ;

**МF-5** Способен применять продвинутые математические методы для решения сложных задач ИИ и разработки новых алгоритмов;

**МF-6** Способен применять логический аппарат для формализации задач представления знаний, проектирования логических моделей и использования систем автоматического доказательства теорем.

    * Книга посвящена детальному изложению круга вопросов, связанных с методом резолюций.
6. https://www.youtube.com/watch?v=vN0oA2z0AW4 - лекция Л. Д. Беклемишева "Компьютерные доказательства" (2021).
    * Лекция известного специалиста по математической логике об огромном пути, проделанном в вопросах изучения проблем математических доказательств с использованием компьютерных систем.
7. https://www.reflexion.ru/club/16-04-09Beklemishev.pdf - презентация к более раннему варианту лекции Л. Д. Беклемишева "Компьютерные доказательства" (2009).
8. Крупский В. Н., Кузнецов С. Л. Практикум по математической логике. СОQ.
    * Система Rocq (до 2025 года - Coq) является программным комплексом, предназначенный для формализации и проверки правильности математических рассуждений. Пособие содержит краткое описание системы Rocq и демонстрирует основные её инструменты, необходимые для содержательного использования.
9. Marker D. An Invitation to Mathematical Logic
    * Свежий (2024) учебник математической логики от одного из крупных специалистов по теории моделей. Делает больше упор на семантику, чем на синтаксис, и показывает связи между логикой и различными разделами математики.
